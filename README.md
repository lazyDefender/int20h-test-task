CLIENT APP - https://int20h-test-task-client.herokuapp.com/
# HOW TO RUN API
    1. cd api (in project directory)
    2. yarn install (or npm install) 
    3. node index.js

# HOW TO RUN CLIENT APP
    1. cd client (in project directory)
    2. yarn install (or npm install)
    3. yarn start (or npm start)
